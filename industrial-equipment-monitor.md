# Equipment Health Monitoring Dashboard

## Overview

A real-time-style industrial equipment monitoring dashboard built using Python Dash and NASA C-MAPSS turbofan sensor data.

This project simulates an operational technology (OT) monitoring interface similar to SCADA systems used in industrial environments.

---

## Features

- Real-time sensor monitoring
- Remaining Useful Life (RUL) visualisation
- Equipment health status detection
- Interactive dashboard interface
- Industrial monitoring simulation

---

## Technologies

- Python
- Dash
- Plotly
- Pandas

---

## Dataset

NASA C-MAPSS Turbofan Engine Degradation Dataset
